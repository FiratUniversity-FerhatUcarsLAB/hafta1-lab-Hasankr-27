Lütfen seçtiğiniz algoritmaya ait çözümü ve diğer isterleri aşağıya ekleyiniz:
ALGORİTMA: MükemmelOmletYap

BAŞLA
    // 1. Malzemeleri tanımla
    YumurtaSayısı ← 3
    TuzMiktarı ← 1 çay kaşığı
    YağMiktarı ← 1 yemek kaşığı
    PişmeDerecesi ← "orta"
    OmletKarışımı ← boş
    OmletYanmadı ← DOĞRU

    // 2. Malzeme kontrolü
    Eğer YumurtaSayısı < 2 ise
        Yaz("Omlet yapmak için en az 2 yumurta gerekir.")
        DUR
    SonEğer

    // 3. Karıştırma işlemi
    Yaz("Yumurtaları kır ve kaseye koy.")
    için i ← 1’den YumurtaSayısı’na kadar TEKRARLA
        Yaz("Yumurta #" + i + " kırıldı.")
    SonTekrarla

    Yaz(TuzMiktarı + " tuz ekle ve iyice çırp.")
    OmletKarışımı ← "Hazır"

    // 4. Tavayı hazırlama
    Yaz("Tavayı ocağa koy.")
    Yaz(YağMiktarı + " kadar yağı tavaya ekle.")
    Yaz("Tavayı ısıt.")
    TavaSıcaklığı ← 0
    while TavaSıcaklığı < 70 derece TEKRARLA
        TavaSıcaklığı ← TavaSıcaklığı + 10
        Yaz("Tava sıcaklığı: " + TavaSıcaklığı)
    SonWhile
    Yaz("Yağ ısındı, tava hazır!")

    // 5. Pişirme süreci
    Eğer OmletKarışımı = "Hazır" ise
        Yaz("Karışımı tavaya dök.")
        PişmeSüresi ← 0

        while PişmeSüresi < 5 dakika TEKRARLA
            PişmeSüresi ← PişmeSüresi + 1
            Yaz("Omlet pişiyor... " + PişmeSüresi + ". dakika")

            Eğer PişmeSüresi = 3 dakika ise
                Yaz("Omleti spatula ile kenarlardan kaldır.")
            SonEğer

            Eğer PişmeSüresi > 5 dakika ise
                OmletYanmadı ← YANLIŞ
            SonEğer
        SonWhile

        Eğer OmletYanmadı = DOĞRU ise
            Yaz("Omlet mükemmel pişti!")
        Aksi halde
            Yaz("Omlet yandı! Bir dahaki sefere süreye dikkat et.")
        SonEğer
    Aksi halde
        Yaz("Karışım hazır değil, işlemi başa sar.")
    SonEğer

    // 6. Servis
    Yaz("Omleti tabağa al ve afiyetle ye 😋")
